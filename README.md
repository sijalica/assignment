City information


This project includes back-end REST services for City information assignment, which was interview requirement. Functionality covers just the required scenarios, with implemented unit tests. This is a JAVA Spring Boot application running with PostgresSQL, while Liquibase has been used as a DB tool for initializing and versioning of DB. Security was implemented using Spring security. Resource server was used with JWT token. For the retrieval of external API, WebClient was used.

Before start:

Create an empty Postgres DB named: cityinformation running on default port 5432.

Start

Run the application, it should start on port 8080.

Postman collection is included here on the repository.
